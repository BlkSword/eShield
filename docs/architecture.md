## ж•°жҚ®еҢ…ж—…зЁӢ

1. еҢ…и§Јжһҗпјҡжңүз•ҢиҜ»еҸ– Eth / IP(v4/v6) / TCP / UDP / ICMP еӨҙйғЁ
2. зҷҪеҗҚеҚ•пјҡLPM Trie жҹҘиҜў CIDRпјҲIPv4 / IPv6пјү
3. з«ҜеҸЈ/еҚҸи®® ACLпјҡжҢүзӣ®зҡ„з«ҜеҸЈдёҺеҚҸи®®еҢ№й…Қ allow/drop и§„еҲҷ
4. йҖҹзҺҮйҷҗеҲ¶пјҡPer-CPU LRU Hash + жҢҮж•°иЎ°еҮҸж»‘еҠЁзӘ—еҸЈ
5. SYN Proxy / SYN Flood / UDP Flood / ICMP FloodпјҡжҢүеҚҸи®®жЈҖжөӢ
6. L7 жү«жҸҸпјҡиҜ»еҸ–еүҚ 64 еӯ—иҠӮиҪҪиҚ·жЁЎејҸеҢ№й…Қ
7. й»‘еҗҚеҚ•пјҡLRU Hash жҹҘиҜўеҲ°жңҹиҮӘеҠЁи§Је°Ғ
8. й»ҳи®Өж”ҫиЎҢпјҡXDP_PASS
